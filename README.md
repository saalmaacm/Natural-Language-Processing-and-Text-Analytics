## Assignment 1

### What’s included
- 📄 [Report](./NLP%20Assignment%201%20Question%201%20%2B%20Report.pdf)
- 📓 [Notebook](./NLP%20Assignment%201.%20Question%202.ipynb)

### What we did
In this assignment, we explored how ambiguity works in language, especially structural ambiguity in sentences. We also used the spaCy library to analyze a dataset of emergency SOS messages.

The analysis included:
- Part-of-Speech (POS) tagging to understand sentence structure  
- Named Entity Recognition (NER) to identify people, locations, and organizations  
- Studying patterns in SOS messages, such as frequent use of nouns and urgency-related words

## Assignment 2

### What’s included
- 📓 [Notebook](./NLP%20Assignment%202.ipynb)
- 📄 [Report](./NLP%20Assignment%202%20report.pdf)

### What I did
In this assignment, we built a text classification system to detect cyberbullying in tweets.

The process included:
- Preprocessing text (tokenization, stopword removal, stemming)  
- Converting text into numerical features using TF-IDF  
- Training and comparing two models:
  - Naive Bayes  
  - Logistic Regression  

Logistic Regression performed slightly better overall, but both models showed a trade-off between precision and recall. This highlighted the challenges of detecting harmful content, especially when language is subtle or context-dependent.
